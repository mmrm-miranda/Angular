#  Angular

Esta carpeta contiene mis desarrollos y prácticas realizadas utilizando el framework **Angular**. Aquí aplico conceptos como enrutamiento, servicios, consumo de APIs externas, manejo de componentes y variables de entorno.

##  Proyectos 

### 1. [Clima](./Clima/)
Una aplicación meteorológica que permite consultar el clima actual de un lugar en especifico.
* **Características:** Consumo de API meteorológica externa para obtener datos en tiempo real.


### 2. [Proyecto (App de Noticias)](./Proyecto/)
Aplicación que muestra las noticias más recientes utilizando la API de NewsAPI.
* **Características:** Muestra titulares, filtrado por categorías y diseño responsivo.


### 3. [Practica 1](./Practica1/)
Proyecto introductorio para sentar las bases en el framework.
* **Características:** Estructura básica de una aplicación Angular.
